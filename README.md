# Notch Apple Rework

This plugin restores the ability to craft **Enchanted Golden Apples** (God Apples) in Minecraft 1.21+. By default, the recipe is balanced for modern gameplay (using Netherite Scraps), but you can change it to anything you want in the configuration.

## Default Recipe

## Commands
- `/notchreload` - Reloads the configuration and updates the recipe instantly.

## Installation
1. Download the `.jar` file.
2. Place it in your server's `/plugins/` folder.
3. Restart the server.
4. Enjoy!
